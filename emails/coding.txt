== Firefox 除錯 ==

首先，Firefox 的基礎架構是 C/C++ / JS / XUL，只要了解 JS 或 CSS 就能幫 Firefox 除錯，在  Mozilla 寫軟體有蠻多面向可以選擇，例如你可以學習 Firefox 或 Thunderbird 的程式設計，主要以 C 或 C++ 為主。其實 Firefox Coding 並不難，對於剛起步的應該可以先從跑流程開始。兩個較大的挑戰是 build Firefox 與 hg version control system。

您可以先參考以下介紹，了解基本概念：
Mozilla Links 正體中文版: 摩茲動手做（一）如何幫 Firefox 抓蟲 http://mozlinks-zh.blogspot.com/2012/02/firefox_25.html

我們社群目前主打 Good First Bug 專案，一步步協助大家嘗試解決 Firefox 的第一個 bug，這是最近我們整理出來的新手 bug 列表：https://etherpad.mozilla.org/moztw-nctu-service-2013 你可以選擇有興趣的 Bug 挑戰。

你還可以上 Bugs Ahoy，查詢你的專長方面有哪些新手 Bug 可以解：http://www.joshmatthews.net/bugsahoy/ 先看看你對什麼比較有興趣，我們可以告訴你更深入的資訊

==網頁開發==

此外，MozTW 在 github ( https://github.com/moztw/central ) ，還有一些網頁、以及其他 HTML5/CSS3 的專案，是可以您可以著手幫忙的部分。Issue 中有一些網站相關的 Bug、與新功能的點子，等你一起來幫忙改。



最後，歡迎訂閱 Mozilla 中文開發者郵件群組：https://lists.mozilla.org/listinfo/dev-general-zh
你可以在此與其他對 Mozilla 開發感興趣的朋友互相交流。關於開發方面的問題也歡迎上 irc.mozilla.org 的 #mozilla-taiwan 頻道發問，這裡有不少 Mozilla 台灣分公司的工程師與社群成員常駐。
