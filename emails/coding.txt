我是 MozTW 的 Thomasy，感謝你對 Mozilla 軟體開發感興趣

首先，Firefox 的基礎架構是 C/C++ / JS / XUL，只要了解 JS 或 CSS 就能幫 Firefox 除錯，在 Mozilla 寫軟體有蠻多面向可以選擇，例如你可以學習 Firefox 或 Thunderbird 的程式設計，主要以 C 或 C++ 為主。其實 Firefox Coding 並不難，對於剛起步的應該可以先從跑流程開始。兩個較大的挑戰是 build Firefox 與 hg version control system。

您可以先參考以下介紹，了解基本概念：
Mozilla Links 正體中文版: 摩茲動手做（一）如何幫 Firefox 抓蟲 
http://mozlinks-zh.blogspot.com/2012/02/firefox_25.html

我們社群目前主打 Good First Bug 專案，一步步協助大家嘗試解決 Firefox 的第一個 bug，這是最近我們整理出來的新手 bug 列表： https://etherpad.mozilla.org/moztw-nctu-service-2013 你可以選擇有興趣的 Bug 挑戰。

你還可以上Bug Ahoy，查詢你的專長方面有哪些新手 Bug 可以解：http://www.joshmatthews.net/bugsahoy/ 先看看你對什麼比較有興趣，我們可以告訴你更深入的資訊

如果您喜歡實驗 HTML5 的最新技術，歡迎挑戰 Mozilla 開發者網路每個月不同主題的 Demo 開發比賽：https://developer.mozilla.org/zh-TW/demos/devderby

此外 MozTW 在 github ( https://github.com/moztw/ ) 有網頁以及其他 HTML5/CSS3 的專案都是可以您可以著手幫忙的部分
