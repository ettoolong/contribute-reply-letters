我是 Mozilla 台灣社群的 Irvin。

首先，歡迎您加入 Mozilla 的大家族。Mozilla 社群是為維護網路的自由與創新的非營利組織，也是 Firefox 的開發者。如果您想要更深入的認識 Mozilla，以下 Youtube 頻道有四部短片推薦您觀看：http://www.youtube.com/playlist?list=PLOHV-5SHjI8hJTZ6rW-cLmPp7tzgI8bji

而我們 Mozilla 台灣社群（MozTW）是一群志同道合、愛好網路的人，為了支持與實踐 Mozilla 宣言（http://www.mozilla.org/about/manifesto.zh-tw.html），聚集在一起貢獻與學習。我們的網站是 moztw.org，任務主要是推廣 Mozilla & Firefox 的理念，及進行正體中文在地化工作（目前我們負責 Firefox 及 Mozilla 相關軟體、網站的中文翻譯）。

目前社群的一些日常工作、以及需要大家貢獻的項目，在我們網站的此頁中有條列整理。您可以看看，並且將您有興趣的部分告知我們：http://moztw.org/contribute

由於正體中文版 Firefox 也是香港主要的語系版本，在此也由香港 Mozilla 社群的 Sammy 向各位介紹：香港 Mozilla 社群是一個義務的社群，一眾志同道合、喜歡互聯網的人，一起為了支持與實踐 Mozilla 理念而付出。歡迎看看以下幾個推廣及集人氣的 Facebook 頁面：http://fb.me/mozillahk、http://fb.me/firefoxhk。香港 Mozilla 社群不定期舉行一些社群聚會，到時歡迎來聊聊啊。 


== 行銷方面 ==

在行銷與企劃方面，我們可提供您發揮的項目有以下幾項：

個人網站、FB 頁面貼紙
	- 請幫助我們幫助我們傳播 Firefox 的消息。請在以下夥伴計畫網站選一個貼紙，然後放到你的網站、部落格，或者任何地方：http://affiliates.mozilla.org
	- Facebook 的 Firefox 夥伴計畫，提供可直接在 FB 分享的個人貼紙：https://www.facebook.com/Firefox/app_373528039377278

抓火狐網站
	- 您可以在 MozTW 社群所建立的「抓火狐」網站建立個人推薦頁面，選擇你最喜歡的 Firefox 功能與推薦套件，取得個人專屬推廣網址，並且貼上專屬貼紙：http://gfx.tw

Marketing Mailing List
	- Marketing 郵件群組是 Mozilla 全球社群間討論行銷事務的主要地方，您可以在這裡先觀看討論： https://groups.google.com/group/mozilla.marketing，然後在這裡訂閱郵件群組：https://lists.mozilla.org/listinfo/marketing

Mozilla 特定活動專案
	- Mozilla 時常會有全球推行的行銷活動，例如 WebMaker Party（http://webmaker.org）與 Firefox Flicks 短片比賽（http://firefoxflicks.org），在這些活動中，常會需要各地的志工協助，相關資訊你可以隨時追蹤 moztw-general 郵件群組：https://groups.google.com/group/moztw-general

台灣在地例行與推廣活動
	- MozTW 大約每一到兩個月舉行一次實體活動，例如各研討會設攤、Firefox 相關 Party、連續聚、推廣講座、紀錄片放映會……等等活動。您可以在這這裡看到過去我們曾舉行過的活動：http://moztw.org/events/
	- 我們的推廣活動企劃目標是具備社群風格的低成本、高歡樂的活動為主。例如去年中在 COSCUP 研討會上的攤位活動計畫（http://bit.ly/YsDdgw）的規劃，結合實體與虛擬，宣傳 Android 版 Firefox 的同時也好玩。歡迎你一起參與活動的籌劃及執行，你也可以提出各種活動的點子，讓我們一起來實現。

產品行銷與活動網站
	- moztw.org 針對 Mozilla 產品會建立專屬的產品頁面，例如 Firefox（http://moztw.org/firefox）、Firefox Android（http://moztw.org/mobile ）。這些產品頁面的設計，需要志願者來收集資料、進行內容規劃，並且與網頁開發者合作開發，上線之後持續進行網頁的曝光度等分析。我們的網站每天大概都有 15k PV 左右，很適合作為學習網路行銷的好標的。相關工作歡迎大家來嘗試與學習。

社交網站行銷
	- MozTW 的 Facebook（http://www.facebook.com/MozTW）、Google+（http://plus.google.com/communities/110692367205504504391/）、Blog（http://mozlinks-zh.blogspot.com/）及 Foxmosa 的三個社交網站頻道（http://moztw.org/foxmosa），都是由社群成員所經營的管道，歡迎您加入來經營內容、也可以嘗試各種行銷的可能性。

如果您有任何問題、對以上任何一個項目感興趣，或者您還不知道可以幫什麼忙，都歡迎直接回信與我聯繫。

也邀請你訂閱我們的電子郵件群組，與社群的大家一起討論、分享您的想法與點子。訂閱之後，您就會開始收到大家互相討論的 email。加入群組之後歡迎發封信跟大家打個招呼喔！https://groups.google.com/group/moztw-general
